# Kids World Wonders Activity Book

**110 pages | Ages 8-12 | Global Landmarks + World Geography**

An explorer-style activity and journal book touring the world's greatest landmarks and natural wonders. Kids learn facts, answer creative challenges, record travel plans, and collect explorer stamps from every continent.

## Book Structure

- **Explorer's Passport + World Map Activity** (8 pages)
- **Wonder Entries x 28** (3 pages each = 84 pages)
- **Continent Challenge Bonus Pages** (10 pages)
- **Design Your Own Wonder** (4 pages)
- **World Explorer Certificate + Back Matter** (4 pages)

## Each Entry Contains

1. Wonder header (name, country, continent, explorer stamp)
2. Fast Facts grid (age, size, builders, cool stat)
3. Explorer Briefing (1 surprising or dramatic fact)
4. Think Like an Explorer (3 creative challenge prompts)
5. My Travel Log (lined journal space)
6. Explorer Checklist (5-6 activity/research challenges)
