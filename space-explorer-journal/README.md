# Kids Space Explorer Journal

**100 pages | Ages 8-12 | Planets, Missions, and Astronaut Challenges**

A mission-log style activity and journal book for young space explorers. Kids complete mission briefings, learn facts about planets and deep-space objects, record their own astronaut logs, and tackle space-themed challenges.

## Book Structure

- **Mission Control Intro + Astronaut Profile Page** (6 pages)
- **Planet + Object Entries x 14** (4 pages each = 56 pages)
- **Famous Space Missions Section** (20 pages)
- **Design Your Own Spacecraft / Colony** (10 pages)
- **Space Explorer Graduation Certificate + Back Matter** (8 pages)

## Each Entry Contains

1. Mission header (object name, type, distance from Earth)
2. Fast Facts grid (size, temperature, moons, cool stat)
3. Mission Briefing (1 key fact written as a mission update)
4. Astronaut Challenges (3 problem-solving prompts)
5. Mission Log (lined journal space)
6. Mission Objectives Checklist (5-6 activity checkboxes)
