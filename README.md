# Kids Activity Book Samples

A collection of activity/journal-style book samples for boys ages 8-12. Each book follows a strict repeatable template system designed for KDP publishing — consistent page structure, fact sections, adventure prompts, journal pages, bucket list checklists, and activity content.

## Template System

Every entry in every book follows the same structure:

```
1. Entry Header (name, location, quick badge)
2. Fast Facts grid (4-6 key facts)
3. Did You Know? (1 surprising fact)
4. Adventure Prompts (3 creative thinking questions)
5. My Journal (lined space for personal writing)
6. Bucket List Challenges (5-6 activity checkboxes)
7. Draw It / Map It (blank creative space)
```

Consistency across 100+ pages is the core design goal. Voice is adventurous and outdoorsy — written for boys who want challenges, not easy answers.

## Writing Approach

- All facts verified against primary sources (NPS.gov, NASA, UNESCO, Britannica)
- Reading level: Grade 4-6 (Flesch-Kincaid)
- AI-assisted drafting with human editing, fact-checking, and consistency review
- Delivered in Google Docs or Word, formatted to client spec
